Essential Keyboard Shortcuts for VSCodium

Here are some of the most useful keyboard shortcuts to enhance your coding efficiency:

Action Shortcut Description
Start/End of Line Home / End Quickly move the cursor to the start or end of the current line.
Find Next Match Ctrl + D Highlight the next instance of a word in your code. Continue pressing this to select additional instances. This is useful for changing variable names.
Skip Instance in Find Next Match Ctrl + K, then Ctrl + D Skip the currently selected instance and move to the next one when using the find next match functionality.
Select All Matches Ctrl + Shift + L Select all instances of a word or phrase in a file, making it easy to refactor code.
Duplicate Code Shift + Alt + Up/Down Duplicate the selected code block or line. Use the up arrow to duplicate above and the down arrow to duplicate below.
Move Code Alt + Up/Down Move a selected block of code or a single line up or down without the need to cut and paste.
Indent Code Ctrl + [ or ] Indent or outdent a selected block of code. Use ] to move it to the right and [ to move it to the left.
Multiple Cursors Alt + Click Add multiple cursors to the code. Click in different locations where you want additional cursors, allowing you to type in multiple places simultaneously.
Add Cursor Above/Below Ctrl + Alt + Up/Down Add a new cursor above or below the current cursor position. This is useful for inserting text in multiple lines at the same position.
Command Palette Ctrl + Shift + P You can use the Command Palette to invoke various extensions and features that might assist with coding.
Switch Between Apps Alt + Tab Switch between open applications. Hold down Alt and continue pressing Tab to navigate through the list of open apps.
