Description Original C language, as defined by Kernighan and Ritchie. First standardized version by ANSI in 1989, known as C89 or ANSI-C. International version of C89, published by ISO in 1990, known as C90. Technical amendment to C90 in 1995, adding wide character support, known as C95. Major revision in 1999, introducing new features like inline functions and boolean types, known as C99. Update in 2011 adding features like _Generic, _Static_assert, and improved multi-core support, known as C11. Revision in 2017 that fixed defects without adding new features, known as C17 or C18. Upcoming major revision with new features and changes, still in draft stage, known as C23 or C2X. GNU's extension to the C language supported by GCC, includes features beyond the official standards. Primarily uses C99 features; some C11 features; relies heavily on GNU C extensions (e.g., `__attribute__` syntax).
