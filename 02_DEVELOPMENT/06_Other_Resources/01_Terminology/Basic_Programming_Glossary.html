Basic Programming Terminology Glossary

Program, Code, Example Code, Application, and Script Glossary

  • Program: A set of instructions written in a programming language that specifies tasks to be performed by a computer.
  • Code: The text written in a programming language that constitutes a program.
    Example Code:
    var name = "John";
    console.log("Hello, " + name + "!");

    Explanation: Defines a variable name with the value "John" and prints a message using the console.log statement.
  • Application: A software program designed to perform specific tasks or provide functionality to users.
  • Script: A program written for a specific runtime environment (e.g., shell script, JavaScript) to automate tasks or provide functionality.

Console, Variables, Objects, Classes

  • Console: A text-based interface for entering commands and interacting with a computer system.
  • Variables: Named storage locations in memory used to hold values that may change during the execution of a program.
  • Objects: Instances of classes in object-oriented programming (OOP) that encapsulate data (attributes) and behavior (methods).
  • Classes: Blueprints for creating objects in OOP, defining attributes and methods that all instances of the class will have.

API (Application Programming Interface)

  • API: A set of rules and protocols that allows one software application to interact with another.
  • Example of API:
    GET /api/users
    POST /api/users
    GET /api/users/{id}
    PUT /api/users/{id}
    DELETE /api/users/{id}

    Explanation: Defines endpoints for interacting with a user management API using HTTP methods.

Micro-service

  • Micro-service: A software architectural style where an application is composed of small, independent services that communicate over a network using lightweight protocols (e.g., HTTP, RPC).
  • Example of Micro-service:
    Service: User Management
    Responsibilities: User CRUD operations, authentication, authorization
    Technology Stack: Node.js, MongoDB
    Communication: REST API, JSON format

    Explanation: Describes a micro-service responsible for user management tasks, implemented using specific technologies and communication protocols.

Statements, Interactive Mode, Code Editors

  • Statements: Instructions or commands written in a programming language to perform specific actions.
  • Interactive Mode: An environment where code can be entered and executed interactively, providing immediate feedback.
  • Code Editors: Software tools for writing and editing source code, providing syntax highlighting, code completion, and other features to assist developers.
